# Proyecto: Día de Canarias

Este es el directorio de desarrollo para la web interactiva del cuadrante del Día de Canarias.

## Estructura del Proyecto
- `index.html` - Página principal de la aplicación.
- `style.css` - Hoja de estilos con el diseño responsive y paleta de colores.
- `app.js` - Lógica interactiva de filtros y buscador.
